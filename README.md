# eslint-config-ydj

## usage

```shell
$ npm install eslint-config-ydj
```

add `.eslintrc` file in your project

copy this code to your `.eslintrc` file

```json

  {
    "root": true,
    "extends": "ydj",
    "env": {
      "browser": true
    },
    "rules": {
    }
  }

```

and then enjoy yourself
